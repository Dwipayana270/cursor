💰 Aljabar di Dunia Keuangan

💸 Pengeluaran Bulanan

Modelkan pengeluaran keluarga:

Total = makanan + transport + listrik + hiburan

Jika makanan = x, transport = 0.5x, listrik = 0.3x, hiburan = 0.2x

Total = x + 0.5x + 0.3x + 0.2x = 2x

📈 Investasi

Modal awal + keuntungan = total investasi

Total = M + r × t

M = modal, r = keuntungan per periode, t = jumlah periode

🏪 Aljabar dalam Bisnis

📦 Penjualan Produk

Model keuntungan penjualan:

Keuntungan = (harga jual × jumlah) - (harga beli × jumlah) - biaya tetap

P = (hj × n) - (hb × n) - Bt

Contoh: Jualan Bakso

Harga jual = Rp15.000/porsi, Harga beli = Rp10.000/porsi

Biaya tetap (sewa) = Rp50.000/hari

Jika terjual 30 porsi: P = (15.000×30) - (10.000×30) - 50.000 = Rp100.000

🏠 Aljabar di Rumah Tangga

⚡ Tagihan Listrik

Total = tarif × pemakaian + biaya administrasi

Jika tarif = Rp1.500/kWh, pemakaian = x kWh, admin = Rp20.000

Total = 1.500x + 20.000

💧 Pengeluaran Air

Total = tarif × volume + biaya administrasi

Jika tarif = Rp5.000/m³, volume = y m³, admin = Rp10.000

Total = 5.000y + 10.000
