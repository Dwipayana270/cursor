📊 Data dalam Bisnis

📈 Analisis Penjualan

Gunakan data untuk mengambil keputusan bisnis:

• Produk terlaris vs kurang laku

• Tren penjualan per bulan

• Demografi pelanggan

Contoh Keputusan:

Stok lebih banyak produk A (penjualan naik 20%)

Berikan diskon produk B (penjualan turun 15%)

💰 Perencanaan Keuangan

Data untuk penganggaran dan perencanaan:

• Pengeluaran rutin vs tidak terduga

• Sumber pemasukan

• Target tabungan/investasi

Contoh Aplikasi:

Alokasikan 30% gaji untuk tabungan berdasarkan data pengeluaran

🏥 Data di Sektor Kesehatan

📊 Monitoring Kesehatan

Data kesehatan untuk gaya hidup sehat:

Data yang Dikumpulkan:

  • Berat badan berkala
  • Tekanan darah
  • Kadar gula darah
  • Aktivitas fisik harian

Visualisasi Data:

  • Grafik berat badan per bulan
  • Diagram lingkaran komposisi makanan
  • Tabel target vs realisasi olahraga

Keputusan Berbasis Data:

Jika berat naik 5% dalam 3 bulan → kurangi porsi makan

Jika tekanan darah > 140/90 → konsultasi dokter

🌍 Data untuk Lingkungan

🌱 Pengelolaan Sampah

Data sampah untuk pengelolaan yang lebih baik:

Jenis Data:

  • Volume sampah per hari/minggu
  • Komposisi sampah (organik/anorganik)
  • Tingkat daur ulang
  • Biaya pengelolaan

Visualisasi:

Diagram batang volume sampah per kategori

Diagram lingkaran komposisi sampah

Aksi Berbasis Data:

Jika sampah organik > 60% → buat kompos

Jika volume sampah naik → edukasi pengurangan sampah

🎯 Proyek Mini: "Data untuk Perubahan"

Lakukan investigasi data untuk masalah di sekitarmu!

  1. Pilih topik (sampah, konsumsi listrik, pengeluaran sekolah, dll)
  2. Kumpulkan data selama 1-2 minggu
  3. Analisis data dengan diagram dan tabel
  4. Buat rekomendasi aksi berdasarkan data
  5. Presentasikan temuan dan rekomendasi

Template Laporan:

1. Latar belakang masalah

2. Metode pengumpulan data

3. Hasil analisis (diagram/grafik)

4. Kesimpulan dan rekomendasi
